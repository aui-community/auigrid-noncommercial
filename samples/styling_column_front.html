rowStyleFunction 으로 결정된 행 스타일(rowStyle)을 항상 행에 대해서만 적용 시킵니다. 속성 rowStyleForOnlyTr = true (기본값: false) 설정 하십시오.

이것의 의미는 TR 태그에만 스타일 적용하라는 의미입니다. 그렇기 때문에 항상 행(row) 스타일은 열(column)보다 하단에 출력 됩니다.

rowStyleForOnlyTr 설정을 하지 않은 경우 TR과 해당 TD 모두에 행 스타일이 적용되어 상단 또는 하단에 CSS 우선 순위에 따라 선택적으로 결정 시킬 수 있습니다.

참고: 칼럼의 styleFunction 으로 결정되는 셀 스타일은 항상 최상단에 출력됩니다.
