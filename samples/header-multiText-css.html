헤더의 텍스트를 개행이 자동으로 이루어지도록 white-space 를 이용한 방법입니다.

그리드 헤더의 white-space 는 기본값으로 nowrap 이 적용되기 때문에 개행이 발생하지 않습니다.

Name과 Country 칼럼은 개행이 이뤄지도록 white-space 를 normal 로 적용한 모습입니다.

// 그리드 속성 설정 var gridProps = { // 헤더 높이 지정 headerHeight : 40, ... // AUIGrid 칼럼 설정 var columnLayout = [ { dataField: "name", headerText: "Name은 이름을 나타냅니다.", headerStyle: "my-header-wrap-style", // white-sapce CSS 적용 하기 위한 헤더 스타일 설정 width: 120 ... /* 헤더 렌더러 스타일 */ .my-header-wrap-style .aui-grid-header-renderer { white-space: normal !important; }
