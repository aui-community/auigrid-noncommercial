가로 병합과 세로 병합을 같이 사용하여 혼합된 병합으로 출력은 불가능합니다. 가로, 세로 병합 중 한가지만 가능합니다.

다음은 가로 병합, 세로 병합을 같이 설정한 경우의 예를 보여주는 데모입니다.

const columnLayout = [{
	dataField: "col1",
	headerText: "COL 1",
	cellMerge: true, // 셀 세로 병합
	cellColMerge: true, // 셀 가로 병합 실행
	cellColSpan: 4, // 셀 가로 병합 대상은 4개로 설정
	width: 100
}, {
	dataField: "col2",
	headerText: "COL 2",
	cellMerge: true, // 셀 세로 병합
	width: 120
}, {
	dataField: "col3",
	headerText: "COL 3",
	cellMerge: true, // 셀 세로 병합
	width: 120
}, {
	dataField: "col4",
	headerText: "COL 4",
	width: 120
}];
