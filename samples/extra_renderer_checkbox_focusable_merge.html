엑스트라 체크박스 렌더러를 출력(showRowCheckColumn = true)한 경우 키보드 방향키로 체크박스로 이동이 가능(focusableRowCheck = true)하도록 포커싱 처리된 예제입니다.

그리드 셀렉션이 존재하는 경우 최좌측 칼럼(대분류)에서 왼쪽 화살표로 엑스트라 체크박스로 포커싱이 가능하며 포커싱이 된 경우 상, 하 방향키로 이동할 수 있습니다.

포커싱된 체크박스에서 스페이스 바(space bar) 키를 누르면 체크 설정/해제가 됩니다.

// 그리드 속성 설정
const gridProps = {
      ...
	// 엑스트라 체크박스 표시 설정
	showRowCheckColumn: true,
	// 행 체크 칼럼(엑스트라 체크박스)의 병합은 grubun1 필드와 동일하게 병합 설정
	rowCheckMergeField: "gubun1",
	// 키보드로 엑스트라 체크 칼럼 포커싱 처리할 수 있는지 여부 (기본값: false)
	focusableRowCheck: true,
};
