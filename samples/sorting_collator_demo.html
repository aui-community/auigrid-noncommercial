JS 표준 API인 Intl.Collator 를 통해 문자열 정렬 비교기를 설정할 수 있습니다.

상단 그리드는 기본 정렬에 해당되고 하단 그리드는 Intl.Collator 를 설정하여 문자열 정렬를 지시한 모습입니다.

■ 기본 그리드

■ 정렬 문자열 비교기 설정 한 그리드

// 정렬 문자열 비교기 설정하기
AUIGrid.setSortCollator(myGridID2, new Intl.Collator('ko', { // 'ko' 설정으로 한글 우선
	sensitivity: 'case',      // 대소문자/악센트 무시
	numeric: true            // 문자열에 있는 숫자를 숫자로 인식 처리
}));
