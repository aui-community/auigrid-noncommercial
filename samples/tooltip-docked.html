그리드의 셀에 마우스가 오면 툴팁(Tooltip)을 표시할 때 고정된 위치에 표시 여부를 지정합니다.(tooltipDocked = true 설정, 기본값 false)

툴팁의 고정 위치는 기본적으로 좌측 하단("bottomLeft")이며, 위치를 지정할 수 있는 tooltipDockedPosition 속성("topLeft", "topRight", "bottomLeft", "bottomRight")을 함께 사용할 수 있습니다.

툴팁의 고정된 위치에 출력될 충분한 공간이 부족한 경우 그리드에 의해 위치는 옮겨질 수 있습니다. 예로 최하단에 툴팁이 출력될 자리가 부족한 경우 상단에 출력됩니다.

// 그리드 속성 설정
const gridProps = {
	//툴팁 출력 여부(기본값: false)
	showTooltip: true,
	// 툴팁 고정된 위치에 출력 시킬지 여부(기본값: false)
	tooltipDocked: true,
	// 툴팁 고정된 위치 설정인 경우 구체적 위치 지정(기본값: "bottomLeft")
	tooltipDockedPosition: "bottomLeft"
};
